# Build Status

## Scope

- This document describes repository behavior, not machine-specific availability.
- For canonical commands, see `docs/FRONTEND_BUILD.md` and `docs/TESTING.md`.

## Current state

- Runtime template first tries `static/dist/app-main.js` and falls back to
  `static/app-main.js` if not found.
- The repository includes `scripts/sync_frontend.py` to mirror source modules to
  `static/*.js` and `static/dist/*.js` when needed.

## When Node is available

Run:

```bash
npm install
npm run frontend:build
```

Then `static/dist/app-main.js` will be generated by Vite from `frontend/src/*.ts`.
